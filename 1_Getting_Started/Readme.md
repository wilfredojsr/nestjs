### Getting Started

Install the nestjs cli

`npm i -g @nestjs/cli`

`nest --version`

`nest --help`

`nest new iluvcoffee`

`cd ./iluvcoffe`

`nest start`

For watching mode run: `npm run start:dev`

The application will start on port 3000

`http://localhost:3000`

Reference: https://docs.nestjs.com
